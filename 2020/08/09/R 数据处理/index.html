R数据处理

基本操作（默认读取到的数据以data命名）：

  • names(data): 查看数据的列名

  • dim(data): 查看行数、列数

  • sum(is.na(data)): 查看缺失值总数

循环读取、操作并保存数据文件

  • 利用paste()函数对字符串进行拼接，首先将所有文件名获取到一个表中，再利用for循环依次读取并执行想要的操作，示例是根据药物名读取药物的ic50数据并导入基因表达数据中

降维方法：
1. 线性方法：常用于保留数据的整体结构

1.1 主成分分析(Principal Component Analysis, PCA):

R语言做PCA分析有 princomp 和 pricomp 函数可以调用,不同的函数需要的数据格式也不相同。pricomp函数只能针对全数据模型有效。

二者不同点 princomp prcomp
mode R-mode Q-mode
基于算法 基于协方差或者相关矩阵提取的特征值 基于奇异值分解

R-mode简单来说就是数据数量要大于变量（因素）数量；而Q-mode正与其相反，简单来讲就是变量数量要多于数据数量。

1.2 对应分析(Correspondence Analysis, CA)

1.3 多重对应分析(Multiple Correspondence Analysis, MCA)

1.4 经典多维尺度分析(classical multidimensional scaling, cMDS)也被称为主坐标分析(Principal Coordinate Analysis, PCoA)

2. 非线性方法：更适合表达数据局部的相互作用关系

2.1 核主成分分析(Kernel Principal Component Analysis, Kernel PCA)

2.2 非度量多维尺度分析(Nonmetric Multidimensional Scaling, NMDS)

2.3 等度量映射(Isomap)

2.4 扩散映射(Diffusion Maps)

2.5 一些包括t分布随机嵌入(t-Distributed Stochastic Neighbor Embedding, t-SNE)在内的邻近嵌入技术

3. 监督降维技术（如果你的观测值带有类别标签，并且你的目标是将观测值分类到已知的与其最匹配的类别中去时，则可以考虑）

3.1 偏最小二乘法(Partial Least Squares, PLS)

3.2 线性判别分析(Linear Discriminant Analysis, LDA)

3.3 近邻成分分析(Neighborhood Component Analysis)

3.4 Bottleneck神经网络分类器
