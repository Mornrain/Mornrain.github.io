R数据处理

基本操作（默认读取到的数据以data命名）：
  • names(data): 查看数据的列名
  • dim(data): 查看行数、列数
  • sum(is.na(data)): 查看缺失值总数
降维方法：
1. 线性方法：常用于保留数据的整体结构

1.1 主成分分析(Principal Component Analysis, PCA):

R语言做PCA分析有 princomp 和 pricomp 函数可以调用,不同的函数需要的数据格式也不相同。

二者不同点 princomp prcomp
mode R-mode Q-mode
基于算法 基于协方差或者相关矩阵提取的特征值 基于奇异值分解

R-mode简单来说就是数据数量要大于变量（因素）数量；而Q-mode正与其相反，简单来讲就是变量数量要多于数据数量。

1.2 对应分析(Correspondence Analysis, CA)

1.3 多重对应分析(Multiple Correspondence Analysis, MCA)

1.4 经典多维尺度分析(classical multidimensional scaling, cMDS)也被称为主坐标分析(Principal Coordinate Analysis, PCoA)

2. 非线性方法：更适合表达数据局部的相互作用关系

2.1 核主成分分析(Kernel Principal Component Analysis, Kernel PCA)

2.2 非度量多维尺度分析(Nonmetric Multidimensional Scaling, NMDS)

2.3 等度量映射(Isomap)

2.4 扩散映射(Diffusion Maps)

2.5 一些包括t分布随机嵌入(t-Distributed Stochastic Neighbor Embedding, t-SNE)在内的邻近嵌入技术

3. 监督降维技术（如果你的观测值带有类别标签，并且你的目标是将观测值分类到已知的与其最匹配的类别中去时，则可以考虑）

3.1 偏最小二乘法(Partial Least Squares, PLS)

3.2 线性判别分析(Linear Discriminant Analysis, LDA)

3.3 近邻成分分析(Neighborhood Component Analysis)

3.4 Bottleneck神经网络分类器

Pytorch学习

A typical training procedure for a neural network is as follows(神经网络的典型训练过程如下)
  • Define the neural network that has some learnable parameters (or weights)
  • Iterate over a dataset of inputs
  • Process input through the network
  • Compute the loss (how far is the output from being correct)
  • Propagate gradients back into the network’s parameters
  • Update the weights of the network, typically using a simple update rule: weight = weight - learning_rate * gradient
常用函数
  • .data: 获得该结点的值，即Tensor类型的值
  • .grad: 获得该节点处的梯度信息
  • grad_fn: 值可以得知该变量是不是一个函数的输出值。若是，则返回一个与该函数相关的对象，否则是None。
常用参数
  • requires_grad: 有 True 和 Flase ，True代表此变量处需要计算梯度，Flase代表不需要；默认为Flase。

TCGA数据类型
1. 临床数据

存储方式为XML格式。

2. 表达数据

表单数据分三种（FPKM、FPKM-UQ、counts）,表达数据的格式为首列为基因id，第二列为表达值。

3. 甲基化数据

甲基化原始数据包含11列信息，第一列为甲基化位点id，第二列为beta value（甲基化程度），第3列至第11列均为该位点的注释信息。

4.microRNA数据

microRNA表达原始数据分为4列，第1列为miRNA id，第2列和第3列分别表示为原始数据和标准化后的reads个数。第4列表示为该miRNA在比对时是否为交叉比对。

5.SNP数据

SNV数据并不是按照不同样本分开存放，整张表格记录了该种疾病所有成对样本经计算后得到的SNV信息。

算法

岭回归（Ridge）
算法适用领域

  • 最先用来处理特征数多于样本数的情况

  • 现也用于在估计中加入偏差，从而得到更好的估计

算法原理
  • 这里通过引入$\lambda$ 限制了所有$\theta^2$之和，通过引入该惩罚项，减少不重要的参数（这个技术在统计学上也叫作缩减）
  • 为了防止过拟合（$\theta$过大），在目标函数$J(\theta)$后添加复杂度惩罚因子，即正则项来防止过拟合。
简单的理解正则化：

  1. 正则化的目的：防止过拟合

    ​ 关于第1点，过拟合指的是给定一堆数据，这堆数据带有噪声，利用模型去拟合这堆数据，可能会把噪声数据也给拟合了，这点很致命，一方面会造成模型比较复杂，另一方面，模型的泛化性能太差了，遇到了新的数据让你测试，你所得到的过拟合的模型，正确率是很差的。

  2. 正则化的本质：约束（限制）要优化的参数

    ​ 关于第2点，本来解空间是全部区域，但通过正则化添加了一些约束，使得解空间变小了，甚至在个别正则化方式下，解变得稀疏了。

正则表达式

  • L1-norm正则
    $$
    J\left ( \theta \right )= \frac{1}{2}\sum_{i}^{m}\left ( y^{i} -\theta ^{T}x^{i}\right )^{2}+\lambda \sum_{j}^{n}\left | \theta _{j} \right |
    $$

  • L2-norm正则

$$
J\left ( \theta \right )= \frac{1}{2}\sum_{i}^{m}\left ( y^{i} -\theta ^{T}x^{i}\right )^{2}+\lambda \sum_{j}^{n} \theta _{j} ^{2}
$$

  • 结合L1-norm、L2-norm进行正则

$$
J\left ( \theta \right )= \frac{1}{2}\sum_{i}^{m}\left ( y^{i} -\theta ^{T}x^{i}\right )^{2}+\lambda(\rho\sum_{j}^{n}\left | \theta {j} \right |+(1-\rho)\sum{j}^{n} \theta _{j} ^{2})
$$
